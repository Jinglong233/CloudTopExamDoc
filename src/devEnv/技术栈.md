# 技术栈

::: tip
有关前端的权限处理，可以参考`Arco Design Pro`官方文档
:::
## 前端技术栈


| 技术栈             | 说明                      |
|-----------------|-------------------------|
| Vue             | 主流的前端开发框架               |
| Arco Design Pro | 开箱即用的前端开发方案             |
| WandEditor      | 很漂亮的编辑器，公式需要自己添加且是Latex |
| dayjs             | 日期工具                    |
| echarts             | 图表                      |
| katex             | 配合WangEditor            |


## 后端技术栈

| 技术栈        | 说明              |
|------------|-----------------|
| SpringBoot | 主流的后端开发框架       |
| MyBatis    | 主流的ORM框架        |
| Redis      | 用于存储Token、个人信息  |
| Xxl-job    | 主要用来处理考试相关的定时任务 |
